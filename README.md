# Beemrus
Beemrus is a free drum machine built right into your terminal!

Written in Python, it is a lightweight, feature-rich drum machine that you can integrate into various other projects, or simply use as a fun creative tool!

## Features
- Load your own samples with Yazi
- Program drum patterns quickly and easily
- Real-time playback and editing
- Fully keyboard based and intuitive navigation

## Controls
  - Arrow keys: move cursor
  - `x`: add hit
  - `-`: remove hit
  - `SPACE`: start/stop
  - `w`: restart sequence
  - `a`: add new samples
  - `q`: quit

## Dependencies
- Python 3.13+
- sounddevice
- soundfile
- Yazi
